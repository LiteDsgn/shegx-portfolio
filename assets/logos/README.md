# Brand logos

Logos for the roles and brands sections. Reference them from `js/main.js`
(role `image` + `logo: true`) or the brands grid in `index.html`.

Naming: kebab-case, matching the brand. Prefer transparent PNG or SVG;
white-on-dark marks sit on the dark logo tiles cleanly.
